Написать консольное приложение, возвращающее значение курса одной валюты к другой.

Приложению передаётся буквенный код валют (USD, RUR, EUR ...).
Ответом приложения является число.

Входные параметры:
 --from - буквенный код валюты, которая является базовой, по-умолчанию BYN
 --to - буквенный код валюты, относительный курс которой требуется получить


Пример:
$  get_course.php --from=USD --to=EUR
1.23
Текущие курсы можно получить из api НБРБ: http://www.nbrb.by/apihelp/exrates
